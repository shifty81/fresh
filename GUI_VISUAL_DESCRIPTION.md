- "Game Settings..." menu item at top

## Visual Style

All ImGui elements use:
- **Font**: Default ImGui font (Proggy Clean)
- **Colors**: Dark theme with blue accents
- **Spacing**: Consistent padding (ImGui defaults)
- **Buttons**: Rounded corners, hover effects
- **Input Fields**: Dark background, light text
- **Sliders**: Blue track, white handle
- **Window**: Dark gray with title bar

## Interactions

**Mouse Sensitivity Slider:**
- Drag left/right to adjust (0.1x to 5.0x)
- Real-time visual feedback of current value
- Camera movement immediately reflects new sensitivity

**Resolution Dropdown:**
- Click to expand list of resolutions
- Select desired resolution
- Click "Apply Resolution" button
- Window smoothly resizes

**World Selection:**
- Single-click to select from list
- Double-click to load immediately
- Keyboard arrows to navigate
- Enter to confirm selection

This GUI system provides a modern, professional interface that matches the quality of contemporary game engines while maintaining the simplicity and clarity that makes it accessible to all users.
