# Terraforming System

The Fresh Voxel Engine includes a comprehensive terraforming system with 10 tools, 3 modes, undo/redo, and support for creating custom structures.

## Tools
- Single Block, Brush, Sphere, FilledSphere, Cube, FilledCube, Line, Flatten, Smooth, Paint

## Modes
- Place, Remove, Replace

## Features
- Undo/Redo (100 operation history)
- Adjustable tool size (1-10)
- Multiple voxel types
- Chunk-aware editing
